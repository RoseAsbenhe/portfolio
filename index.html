ABOUT ME

hi! my name is vanessa böhner and I'm a frontend developer based in munich.
since I was twelve I'm interested in webdesign and that's why I wanted to study computer science. but of course it has to be something cool with media. so I've got my bachelor degree in mediainformatics combined with human computer interaction in 2013, followed by my master's degree at the ludwig-maximilians university in munich in 2015. (Thesis: Enhancing Trust in Verified Websites through Individual Browser Themes)

WHAT I DO

digital products in the web have to be designed and developed for the best user experience. that's what I do.

web development

I build websites in terms of the look and feel and how users will interact with them across all common browsers and devices. I use technologies like HTML5, LESS / SASS, ES6 / Vanilla Javascript / Coffeescript, Underscore, Backbone.js, Gulp and Node.js.

interaction design

web design is more than just writing some code. my focus is on the usability of the page for a perfect user experience (UX). therefore I use techniques of the human computer interaction for the user interface (UI)

WHAT I HAVE DONE

Why do I use JavaScript? I've got two answers for you. First: I haven't got a choice. The Web is my passion and JavaScript the only available language - Java has never succeeded in this environment. And yeah.. I don't like Java anyway. Second: JavaScript is an amazing language!
